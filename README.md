# UI Automation – Personal Blog QA

## 📌 What This Is
End-to-end UI automation tests written in Cypress for my personal blog. Focused on validating navigation, layout consistency, and basic accessibility.

## 🧪 QA Focus
- Manual test cases for core flows (navigation, search, post view)
- UI automation using Cypress
- Visual layout validation and regression scenarios
- Accessibility check using `cypress-axe`
